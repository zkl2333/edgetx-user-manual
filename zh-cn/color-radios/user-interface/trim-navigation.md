# Trim导航

在NV14和EL18遥控器上，可以使用Trim帽开关导航菜单选项。

在无线电设置屏幕中，您可以使用以下选项之一配置**帽子模式**：

* **仅修剪**：Trim帽开关仅用于调整修剪值。
* **仅按键**：Trim帽开关用于导航菜单选项（如下所述）
* **可切换**：Trim帽开关功能可以在**修剪**和**按键**之间动态切换。

### 动态切换模式

1. 将**帽子模式**配置为**可切换**。
2. 按住**左帽**。
3. 立即按下**右帽**。

{% hint style="info" %}
当无线电上电并且帽子模式设置为**可切换**时，初始模式将始终设置为**修剪**。
{% endhint %}
