# 无线电设置

**无线电设置** 部分包含了配置您的无线电的所有选项。在页面顶部，您会看到一些图标，当选中时，它们将带您到不同的无线电设置页面。无线电设置的默认屏幕是 [工具](tools.md) 屏幕。

无线电设置屏幕顶部的图标包括（从左到右的顺序）：

- [工具](tools.md)
- [SD 卡](sd-card.md)
- [无线电设置](radio-setup/)
- [主题](themes.md)
- [全局功能](global-functions.md)
- [训练者](../model-settings/model-setup/trainer.md)
- [硬件](hardware.md)
- [版本](version.md)
